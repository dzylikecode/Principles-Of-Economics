# The Cost of Something Is What You Give Up to Get It

由于人们面临着权衡取舍，所以做出决策就要比较可供选择的行动方案的成本与利益。

## example

那些大学里的运动员如果退学转而从事职业运动，就能赚几百万美元。他们深深认识到，他们上大学的机会成本极高。所以他们通常决定：不值得花费这种成本来获得上大学的利益。

> 此时学习的成本是本来可以赚到的几百万美元
